# Mobile Game Analytics Project

## Overview
This project analyzes player engagement, retention, gameplay activity, and monetization trends for mobile games using Python, SQL, Excel, and Power BI.

The goal of this project was to identify gameplay patterns, track KPIs, and generate insights that can help improve player engagement and game performance.

---

## Tools & Technologies
- Python
- Pandas
- SQL
- Power BI
- Excel
- Jupyter Notebook

---

## Key Metrics Analyzed
- Daily Active Users (DAU)
- Monthly Active Users (MAU)
- Session Length
- Retention Rate
- In App Purchase Trends
- Player Engagement
- Gameplay Activity
- Revenue Trends

---

## Project Workflow
1. Data collection and cleaning
2. Data validation and preprocessing
3. KPI analysis
4. Dashboard creation
5. Trend and engagement analysis
6. Reporting and visualization
